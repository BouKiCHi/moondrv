
　　PCMTOOL説明書 160229


■説明

　PCMTOOLは、入力テキストファイルより、wavファイルやパラメータの指定を行い、
PCMファイルを作成するツールです。


■使用方法

　データの作成は、Windows等のコンソール上で、

>PCMTOOL (PCM定義テキスト)

と入力します。（ファイル名は任意）

データの読み出しは、MSX上のDOSで、
>PCMLOAD (PCMファイル)
と入力します。

データの読み出しには、MDRファイルにパッキングを行う方法もあります。


■PCM定義テキストについて

　PCM定義テキストは、以下の項目を順番に並べた物です。

file.wav bit loop end
lfo vib ar d1r dl d2r rc rr am


・file.wavは、WAVファイルを指定します。
フォーマットによっては読めない場合があるので変換を行う必要があるかもしれません。


・bitは生成されるデータのビット長で、指定できる数値は以下の通りです。

0 = 8bit
1 = 12bit
2 = 16bit


・loopとendの数値は「指定サンプル数-1」です。
loopとendは、PCMTOOLの内部でアドレスに変換されます。

計算式は以下の通りです。

　実際のバイト数 = サンプル数 * (ビット長/8)


　lfoなどはそれぞれのレジスタにある数値と同じです。

注記: LFO VIBなどの各種パラメータは音色テーブルで
設定するのですべて0で埋めてもよいと思われます。


■定義されたPCMの使用方法

　MMLからアクセスするには、@TONEの音色番号を$180(= 10進数で384)以降
にすることで音色を利用することができます。

MMLでは次のようになります。
----------------------
@TONE0 = {
 $00, $7f, $0180, 6000,  $20, $f2, $13, $0f, $00
}
----------------------


■実際のサンプリング周波数とピッチ補正値（参考）

　ピッチ補正値は、MML上の定義の4番目の数値になります。


数値はそれぞれo4c時
11KHz = 3000
16KHz = 4000
22KHz = 4500
44KHz = 6000
